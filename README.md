# Quiz

Quiz is a Online q&a platform with Flask.

在线问答平台

## Installation

Make sure you have installed [pipenv](https://docs.pipenv.org/en/latest/)

```bash
pip install pipenv
```

Then install libs

```bash
pipenv install
```

## Usage

```bash
flask run
```
